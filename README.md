## 说明
### recyclerview的应用

```思想
组合优于继承的是设计方式的适配器
```

### LoadingViewHelper架构(由LuckyJayce的MVCHelper修改演变而来)
```优点
* 继承了MVCHelper的优点
* 实现下拉刷新，滚动底部自动加载更多，分页加载，自动切换显示网络失败布局，暂无数据布局，,真正的MVC架构.
* 自定义多个泛型参数传值
* 支持SwipeRefreshLayout,in.srain.cube:ultra-ptr等下拉刷新组件
* 支持android-async-http和volley等网络请求架构
```
